## ✨ What is WordKeeper?

**WordKeeper** is a personal foreign-word dictionary for language learners who want a simple place to collect vocabulary and return to it later.

Instead of keeping words across notes, chats, screenshots, and browser tabs, WordKeeper gives you one clean workspace for:

- creating personal dictionaries;
- saving words, translations, parts of speech, and comments;
- using automatic translation suggestions;
- reviewing words with focused practice sessions;
- practicing through the Telegram bot.
